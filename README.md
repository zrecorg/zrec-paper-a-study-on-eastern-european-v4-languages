# Dataset and Code for: Tuning of Language Models in Eastern European Languages on Twitter/X

 

**Article DOI:** 10.5281/zenodo.17723755 

PDF: https://ceur-ws.org/Vol-4092/paper19.pdf

 

This repository contains the dataset and code used in the paper:

 

**Filip, T., Pavlíček, M., & Sosík, P. 

*Tuning of language models in Eastern European languages on Twitter/X.* 

In Proceedings of the Workshop on Artificial Intelligence and Language Technologies (Vol. 4092). CEUR-WS.**

 

The dataset includes the materials necessary to reproduce the experiments described in the publication, 

including preprocessing scripts, evaluation data and supporting resources.
